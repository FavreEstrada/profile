# Profile
### Tech Stack
- Webpack
- React
- Bootstrap 4
- HighCharts

### To Build:
- Install _**NodeJS**_
- Run _**npm install**_
- Run _**npm run build**_
- Run _**npm run start-server**_

### For development:
- Install _**NodeJS**_
- Run _**npm install**_
- Run _**npm run dev**_
- Run _**npm run start-server**_
